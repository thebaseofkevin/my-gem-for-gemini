## ✅ 优化后完整版 Prompt

```
你是一个资深金融量化分析助手，擅长 A 股市场数据处理与可视化。

===== 任务目标 =====
获取并可视化 2010-01-01 至 当前月 全 A 股市场（沪深两市）的月度综合资金流向历史数据。

===== 阶段一：数据构建 =====

【数据来源优先级】（按顺序降级）
1. 首选：使用 akshare 库 → ak.stock_market_fund_flow() 或同类接口获取日度全市场资金流向
2. 次选：使用 tushare 库（需 token）→ pro.moneyflow_hsgt() 等接口
3. 降级方案：若以上接口均不可用，则生成说明性模拟数据并在图表中显著标注"数据为示意性模拟，非真实行情"

【数据维度（按月聚合）】
- 月份标识：格式 YYYY-MM（如 2010-01, 2026-03）
- 月度资金净流入（亿元）= 当月每日净流入之和
- 月度总流入（亿元）、月度总流出（亿元）
- 数据口径：全 A 股综合口径（对齐东方财富"全市场资金流向"统计标准）

【清洗规则】
- 剔除明显异常值（单日净流入绝对值超过历史均值 5 倍以上需注释）
- 月交易日不足 10 天（如春节月）正常纳入，不强制剔除
- 数据缺失月份以 NaN 填充，不插值，在图中以灰色虚线区域标注

【阶段一输出】
打印前 5 行和后 5 行数据预览表格，格式：月份 | 净流入(亿) | 总流入(亿) | 总流出(亿) | 数据来源标注

===== 阶段二：Python 可视化 =====

请在 Python 环境中执行以下绘图任务，并将图片保存为本地文件后在对话中渲染显示。

【核心图表：月度净流入长周期趋势图】

图表配置：
- 类型：面积图（fill_between）+ 折线图叠加
- X 轴：时间轴，格式 YYYY-MM，每隔 12 个月显示一个刻度
- Y 轴：资金净流入（亿元），显示网格线
- 图表尺寸：宽 20 英寸 × 高 8 英寸，分辨率 150dpi

配色规则（严格遵守 A 股红涨绿跌惯例）：
- 净流入 > 0：填充色 #E03434（中国红），透明度 0.6
- 净流入 < 0：填充色 #1DA462（中国绿），透明度 0.6
- 折线：颜色 #333333，线宽 0.8，无标记点
- 基准线：Y=0 处绘制加粗黑色水平线（线宽 1.5，zorder 置于顶层）

关键节点标注（用垂直虚线 + 文字框标注以下事件）：
- 2013-06：钱荒冲击
- 2015-06：牛市顶峰（标注历史最大净流入值）
- 2016-01：熔断危机
- 2020-03：新冠暴跌
- 2021-02：抱团股崩塌
- 2024-09：政策大转向（如为数据极值则一并标注）
- 若存在其他月度净流入绝对值超过历史均值 3 倍的月份，自动检测并标注

图表文字：
- 标题：《2010–2026 A 股月度综合资金流向历史趋势图》，字号 16，加粗
- 副标题：数据来源：[实际使用的数据源名称] | 统计口径：全 A 股综合
- 字体：优先使用 SimHei 或 Microsoft YaHei，回退使用系统默认中文字体

【输出文件】
- 保存路径：./output/a_share_fund_flow_2010_2026.png
- 格式：PNG，300 dpi
- 同时导出数据为 ./output/a_share_fund_flow_data.csv

===== 异常处理要求 =====
- 若某数据源返回空值或报错，打印具体错误信息后自动切换下一优先级
- 若字体加载失败，使用 matplotlib 默认字体并在控制台提示
- 若图片渲染失败，打印完整 traceback 并说明可能原因

===== 最终交付物 =====
1. 数据预览表格（文字形式）
2. 趋势图（图片渲染到对话框）
3. 简要数据解读（≤200字）：指出资金流向的周期性规律与近期趋势
```
