## [Phase 2] 动态全息估值

### 2.1 估值工具选择矩阵

| 标的类型 | 主估值工具 | 辅助工具 | 禁用工具 |
|---------|-----------|---------|---------|
| 科技/硬件 | 远期FCF年化×动态倍数 | PS（成长期） | 静态PE |
| 医药/Biotech | 所有者盈余+管线rNPV | EV/EBITDA | PE（亏损期） |
| 消费/品牌 | DCF+品牌溢价修正 | EV/EBIT | PS |
| 金融 | PB+ROE均值回归 | 股息折现 | EV类 |

### 2.2 动态乘数修正（强制执行）

| 行业周期位置 | 允许估值倍数 | 逻辑 |
|------------|------------|------|
| 爆发扩张早期 | 25x – 35x FCF | 稀缺性溢价 |
| 扩张中期 | 18x – 25x FCF | 合理成长定价 |
| 景气顶点 | **强制压缩至 12x–15x** | 周期均值回归 |
| 下行/衰退期 | 8x – 12x FCF | 防御性折价 |

### 2.3 资产负债清洗
```
净现金 = (货币资金 + 交易性金融资产) - 有息负债合计
含金量EV = 市值 - 净现金（净现金为负则加回）
核心EV/FCF = 含金量EV ÷ 年化自由现金流
```

### 2.4 三档价格体系

| 档位 | 计算逻辑 | 含义 |
|------|---------|------|
| 🟢 保守安全价 | 基于悲观FCF假设×12x | 闭眼买入线 |
| 🟡 合理估值价 | 基准FCF×行业中轴倍数 | 正常买入线 |
| 🔴 高估警戒价 | 乐观FCF×顶部倍数 | 开始减仓线 |

## [Phase 3] 威科夫量价图谱（操作级）

### 3.1 大周期阶段识别
标注当前处于哪个阶段，并给出**判断依据**（不能只贴标签）：

```
积累期          标注期          派发期          下跌期
PS→SC→AR  →  ST→Spring→LPS  →  BC→UT→SOW  →  LPSY→下跌
```

### 3.2 主力足迹信号库

| 信号名称 | 量价特征 | 操作含义 |
|---------|---------|---------|
| Spring | 跌破支撑+快速收复+量能萎缩 | 潜在买点，等BU确认 |
| LPS | 回调不破前低+量能递减 | 积累期末，可建仓 |
| BU (Back Up) | Spring后回踩+缩量 | Spring买点的确认信号 |
| UT (上冲回落) | 突破前高+放量滞涨+快速回落 | 警惕派发，减仓信号 |
| SOW | 下跌放量+反弹无力 | 确认派发，清仓信号 |
| BC (买入高潮) | 巨量长上影 | **禁止追涨** |

### 3.3 共振买卖纪律（刚性规则）

**✅ 完美买点**（必须同时满足）：
1. 价格处于【保守安全价】以下
2. 威科夫阶段处于【Spring】或【LPS】
3. Spring需等待【BU缩量确认】后才入场

**🚨 强制离场**（满足任意一条）：
1. 价格超过【高估警戒价】20%以上
2. 威科夫出现【UT放量滞涨】+ 随后【SOW】确认
3. Phase 1排雷清单中任意硬性指标恶化

## [Phase 5] 三年回报映射

```
目标价格（基准情景） → 预期年化回报
目标价格（乐观情景） → 预期年化回报  
目标价格（悲观情景） → 预期年化回报

Expected Return = (P_target / P_current)^(1/3) - 1
```

同时计算**盈亏比**：上行空间 ÷ 下行风险（至保守安全价距离）
> 盈亏比 < 2:1 的标的，**无论多看好，仓位不超过3成**。
